# Help Desk Ticket System

A beginner Java console application that allow users to create, view and update help desk tickets.

## Feaures
-Create tickets with description and priority
-View all tickets
-Update Ticket status
-Simple console menu system

## Technologies used
-Java
-ArrayList
-Object-Oriented Programming
-Git & Github

## How to run 
compile:
javac *.java

Run:
java Main
